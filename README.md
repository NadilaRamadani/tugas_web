# Website Profil Pribadi

Website ini dibuat untuk tugas membuat web menggunakan HTML, CSS, JavaScript, dan Laravel.

## 👩‍💻 Tentang Website
Website ini berisi:
- Halaman Home
- Profil diri
- Riwayat pendidikan (SD, SMP, SMA, Kuliah)
- Cerita pribadi
- Galeri foto
- Video
- Kontak

## 🛠️ Teknologi yang Digunakan
- HTML
- CSS
- JavaScript
- Bootstrap
- Laravel

## 📁 Struktur File
- home.blade.php → halaman utama
- profil.blade.php → profil diri
- cerita.blade.php → cerita pribadi
- foto.blade.php → galeri foto
- video.blade.php → video
- kontak.blade.php → halaman kontak

## 📸 Fitur
- Tampilan menarik
- Navigasi mudah
- Responsif (bisa dibuka di HP/laptop)
